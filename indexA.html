Project Abstract

Our teams objective is to create a handheld water depth sensor that will interface with a phone to give the user the depth, for use in applications such as canyoneering, fishing, cliff diving, search and rescue.

We need to utilize either ultrasonic sensors, pressure sensors, or sonar in order to calculate water depth regardless on how clear the water is. Needs to be small enough to be handheld, lightweight for taking in a pack.

Importance

This project allows improved safety in several applications including those mentioned above. Knowing how deep or shallow water is can prevent accidents from happening in areas.

Challenges

Hardware
  • Reliable power supply
  • Designing a pcb to manage the sensor and connect with bluetooth
  • Waterproofing
Software
  • Bluetooth interfacing with a phone
  • Mobile app development
  • Correct calculations of water depth from sensor input

Proposed Methods

Hardware
  • Build a rechargable lithium-ion battery pack that can handle the requirements of the device
  • Study the datasheets of the sensor we use in order to interface it correctly with a microcontroller
  • Design the enclosure such that it protects the electronics from water and debris
Software
  • Learn Bluetooth protocols from online sources
  • Either recruit someone who knows app development or one person dedicates time to learn this
  • Navigate the datasheet of the sensing device or board
